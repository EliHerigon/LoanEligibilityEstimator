# Loan Eligibility Estimator (Angular + .NET)

A small full-stack demo app that lets a user enter income, debts, credit score, and loan details, then returns an estimated monthly payment, DTI%, credit tier, and an eligibility decision.

## Tech
- Angular (TypeScript) frontend
- .NET minimal API backend

## Run locally

### 1) Backend (API)

cd backend/BackendApi  
dotnet run  

API runs at: http://localhost:5100  

### 2) Frontend (Angular)

cd frontend  
npm install  *Only need to do this step the first time running*
npm start  

App runs at: http://localhost:4200
